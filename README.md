# appsticlanding

صفحة تحميل تطبيق **الاحصاء الطبي** لأنظمة Android و iOS.

## المحتويات

- `index.html` — صفحة الهبوط
- `downloads/` — ملفات التطبيق (APK و mobileconfig)
- `images/` — شعار التطبيق والصور
- `server.py` — خادم محلي للتطوير والاختبار

## التشغيل المحلي

```bash
python3 server.py
```

ثم افتح: http://localhost:8080

## الروابط

- تحميل أندرويد: `/downloads/appstic.apk`
- تحميل آيفون: `/downloads/al-ehsaa-altibbi.mobileconfig`
